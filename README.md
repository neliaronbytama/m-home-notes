# m-home-notes

一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的页面彼此独立，不依赖任何特定域名或网站环境，适合作为静态内容存档或临时演示用途。

## 项目简介

本仓库的主要目的是整理和保存各类独立的 HTML 页面。这些页面可能是笔记、工具界面、原型设计或信息展示页，均以纯前端方式构建，无需后端服务支持。仓库本身不面向特定用户群体，也不包含任何商业或推广目的。

## 目录说明

```
m-home-notes/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的公共资源（样式、脚本、图片等）
├── index.html      # 仓库首页，列出所有页面索引
└── README.md       # 本说明文件
```

- **pages/**：每个子目录或文件代表一个独立页面，命名清晰，便于区分。
- **assets/**：公共资源文件，可被多个页面共享引用。
- **index.html**：提供页面列表，方便快速浏览和访问。

## 页面归档说明

归档的页面均为完整的 HTML 文件，包含必要的样式与脚本。建议遵循以下原则：

- 每个页面应保持自包含，尽量减少外部依赖。
- 如需引用外部资源（如 CDN 库），请在页面内注明来源。
- 页面文件名应反映内容主题，避免使用无意义编号。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的链接。
- 更新页面：直接替换对应文件，注意保持文件名一致。
- 删除页面：移除文件及 `index.html` 中的对应条目。
- 公共资源：如需添加新资源，请放入 `assets/` 并注明用途。
